# Manual de MPV Media Player - Kael OS

MPV es un reproductor multimedia altamente eficiente y minimalista que se controla principalmente mediante atajos de teclado. Está integrado de forma nativa en Kael OS para reproducir tus archivos de audio y video directamente desde Yazi.

---

## 📂 Archivos de Configuración
*   **Configuración General:** [`~/.config/mpv/mpv.conf`](file:///home/jose/.config/mpv/mpv.conf) (Configura reproducción sin bordes gráficos, volumen inicial al 80%, aceleración por hardware y búsqueda inteligente de subtítulos locales).

---

## ⌨️ Atajos de Teclado Fundamentales

### 1. Control de Reproducción
*   `Space` / `p` → Pausar / Reanudar la reproducción del video.
*   `f` → Alternar pantalla completa (Fullscreen).
*   `q` / `Esc` → Detener la reproducción y cerrar MPV.

### 2. Navegación de Tiempo (Seek)
*   `Flecha Izquierda` / `Flecha Derecha` → Retroceder / Avanzar 5 segundos.
*   `Flecha Abajo` / `Flecha Arriba` → Retroceder / Avanzar 1 minuto.
*   `Shift + Flecha Izquierda` / `Shift + Flecha Derecha` → Retroceder / Avanzar 1 segundo de manera precisa.

### 3. Control de Audio y Volumen
*   `9` / `0` → Disminuir / Aumentar el volumen del audio.
*   `m` → Silenciar / Activar el audio (Mute).
*   `Ctrl + +` / `Ctrl + -` → Sincronizar o retrasar la pista de audio con el video.

### 4. Subtítulos y Velocidad
*   `v` → Activar / Desactivar subtítulos.
*   `j` / `J` → Ciclar entre los archivos de subtítulos disponibles.
*   `z` / `x` → Sincronizar / Desfasar el tiempo de aparición de los subtítulos.
*   `[` / `]` → Disminuir / Aumentar la velocidad de reproducción.
